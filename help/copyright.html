This software comes with a few free libraries which are included in the main jar file in binary (class) format. In case of the Eisenkraut source distribution, they appear as separate jar files in the libraries folder. Their sourcecode is publicly available on the net or -- in consonance with §6 b of the LGPL -- on request through the author. these libraries are:

  • JCollider which provides the infrastructure to control SuperCollider. GPL.
  • JCollider itself uses the NetUtil OSC library (LGPL)
  • MRJAdapter by Steve Roy, which improves Mac OS look-and-feel. Artistic License

Eisenkraut makes use of the following third party source-codes:

  • BrowserLauncher by Eric Albert, used to open the system's default web browser.

General Prefs

General preferences affect the overall program look and performance.

  • Temporary Folder
  • Waveform Cache (Active, Capacity, Folder)
  • Reveal File Command: Specify a shell command to execute when pressing the magnifying glass icon in the bottom left of a document frame. For example, on Linux you might want to use xdg-open %d to show the directory in the desktop manager, or nautilus %p to show the file in the Nautilus file manager. Allowed escape words are %p (full path), %d (directory), %f (file name). Use capitalized character (e.g. %P) for URL style.
  • Look-and-Feel (Look-and-Feel class, Window Decoration, Internal Frames, Grow Box, Floating Palettes)

Audio File Header

a tiny palette window allowing you to change the sample rate and commentary meta-information of the current sound file. This palette can be opened with one of the document's audio info buttons.

the sample rate change is a mere "header change", it will not apply a resampling process. use this to correct sample rate information or to temporarily play back the file at a different speed and pitch.

Note: Currently, commentary information is only stored in AIFF and Snd/AU files. It is lost when saving to any other audio format.

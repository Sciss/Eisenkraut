Audio Prefs : SuperCollider Synth Application (scsynth)

the path specifies the synthesis server which is run as a unix command when pressing the boot button of the main window. it must point to the file scsynth[.exe] in your supercollider folder! the path is usually something like "/usr/bin/scsynth" or "/Applications/SuperCollider_f/scsynth".

note that traditional computers are not shipped with supercollider pre-installed, so you may have to install supercollider first before using eisenkraut. the main supercollider site is www.audiosynth.com. downloads can be found for example on sourceforge.net/projects/supercollider.

HTML Help Display

Help documents are shown in this display area. Normal blue links connect to other local documents. Italic links connect to internet sites. Purple links scroll down the current document. Use the 'Back' button to return to the previously shown page. Use the 'Open in Browser' button to view the current document in a web browser.

The special button 'Open with jEdit' is there as an easy way for me to update the docs. It requires Mac OS X and jEdit being installed.

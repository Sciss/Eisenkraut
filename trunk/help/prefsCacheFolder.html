General Prefs : Cache Folder

sets the folder in which waveform caching files are stored. note that unlike other programmes, eisenkraut doesn't save the cache files in the same place as the soundfiles themselves, so you don't end up with your harddisk cluttered with ".ovr" files or the like. you can at any time empty the cache folder.

by default, the folder Eisenkraut/cache inside your home folder is used. files are named using a hash code and therefore their names are not human readable.

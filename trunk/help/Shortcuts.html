Keyboard Shortcuts

Note: the shortcuts have been mainly designed on Mac OS with German Apple Pro and MacBook keyboards. There may be inconsistencies with other configurations. Please report any problems and I try to fix them.

The key modifier Meta is equivalent to the Apple (or Command) key on Mac OS, and Ctrl+Shift on Linux and Windows.

Document Frame

   Transport:

  • Space or Numpad-0 : toggle play / stop
  • Ctrl+Space : play selection
  • Shift+Space : play at half speed
  • Alt+Shift+Space : play at double speed
  • Numpad-/ : toggle loop playback
  • Numpad-1 and 2 : rewind and fast forward

   General Editing:

  • F1 ... Fn : select tool
  • Escape : cancel tool gesture (depends on selected tool)
  • Caps lock : toggle blending mode
  • V : toggle catch mode

   Selection and Navigation:

  • Return : stop playback and go to start of document
  • Shift+Return : extend selection to document start
  • Alt+Shift+Return : extend selection to document end
  • Cursor Left/Right : scroll to begin/end of selection
  • Cursor Up/Down : set timeline position to selection begin/end and clear selection
  • Alt+Cursor Up/Down : set timeline position to selection begin/end
  • Tab : select next region between adjectant markers
  • Alt+Tab : select previous region between adjectant markers
  • Shift+Tab : extend selection to next marker
  • Alt+Shift+Tab : extend selection to previous marker
  • Alt+F : fit visible span to selection
  • Delete (Backspace) : remove selected time span
  • Meta+A : select whole document span
  • M : drop marker at current timeline position. only effective if marker display is enabled from the view menu.
  • G : open go-to-time dialog

   Zooming:

  • Ctrl+Cursor Left/Right : horizontal zoom out/in
  • Alt+A or Meta+Cursor Left : zoom all the way out
  • Meta+Cursor Right : zoom to sample level
  • Ctrl+Cursor Up/Down : vertical zoom out/in (or ceiling adjustment in logarithmic mode)
  • Ctrl+Alt+Cursor Up/Down : noise floor adjustment (in logarithmic mode)

Recorder Dialog

these are subject to changes!

  • Meta+R : start recording
  • Meta+Return : stop and commit recording
  • Meta+Period : abort recording
  • Escape : close dialog

Special Gadgets

   Path Fields:

  • Meta+Alt+Numpad-1 to 9 : store directory in user preferences 1 ... 9
  • Meta+Numpad-1 to 9 : recall directory from user preferences 1 ... 9

   Parameter Fields:

  • Escape : unfocus gadget

   Confirming Unsaved Changes:

  • Meta+D : choose "don't save"
